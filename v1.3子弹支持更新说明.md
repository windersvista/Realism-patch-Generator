# v1.3 子弹支持更新说明

## 更新时间
2026年2月20日

## 更新内容

### ✅ 子弹数据识别和生成

现实主义补丁生成器已升级至 **v1.3**，现已完全支持子弹（弹药）数据的识别和补丁生成。

## 主要功能

### 1. 自动识别子弹
- 通过 `parentId: 5485a8684bdc2da71d8b4567` 自动识别子弹物品
- 支持STANDARD和VIR两种数据格式
- 无需额外配置或模板文件

### 2. 子弹属性提取
脚本会自动从原始数据中提取以下子弹属性：

#### 基础属性（总是生成）
- **Damage**: 伤害值
- **PenetrationPower**: 穿透力
- **LoyaltyLevel**: 忠诚度等级（默认1）
- **BasePriceModifier**: 基础价格修正（默认1）

#### 高级属性（如果存在）
- **InitialSpeed**: 初始速度（米/秒）
- **BulletMassGram**: 子弹质量（克）
- **BallisticCoeficient**: 弹道系数

### 3. 独立补丁文件
子弹补丁单独保存为 `ammo_realism_patch.json`，便于管理和使用。

### 1. 子弹识别逻辑
```python
def is_ammo(self, parent_id: str) -> bool:
    """判断是否为子弹"""
    return parent_id == "5485a8684bdc2da71d8b4567"
```

### 2. 子弹补丁生成
```python
def create_default_ammo_patch(self, item_id: str, item_info: Dict) -> Dict:
    """创建默认的子弹补丁"""
    patch = copy.deepcopy(DEFAULT_AMMO_TEMPLATE)
    patch["ItemID"] = item_id
    
    # 提取Name
    if item_info.get("name"):
        patch["Name"] = item_info["name"]
    else:
        patch["Name"] = f"ammo_{item_id}"
    
    # 提取子弹属性
    props = item_info.get("properties", {})
    if "Damage" in props:
        patch["Damage"] = props["Damage"]
    if "PenetrationPower" in props:
        patch["PenetrationPower"] = props["PenetrationPower"]
    # ... 更多属性
    
    return patch
```

## 注意事项

### 1. 子弹识别
- 只有 `parentId` 为 `5485a8684bdc2da71d8b4567` 的物品会被识别为子弹
- 其他类型的物品（如手榴弹）不会被处理

### 2. 属性提取
- 如果原始数据中缺少某些属性，将使用默认值
- 默认 Damage: 50，PenetrationPower: 20
- 高级属性（InitialSpeed等）仅在原始数据包含时才添加

### 3. 兼容性
- ✅ 完全向后兼容v1.2、v1.1和v1.0
- ✅ 不影响武器和配件补丁生成
- ✅ 可以与VIR和STANDARD格式混合使用

## 常见问题

### Q1: 子弹补丁文件为空？
**A**: 检查Items文件夹中是否有parentId为`5485a8684bdc2da71d8b4567`的物品。

### Q2: 子弹属性值不正确？
**A**: 检查原始数据中的`Damage`、`PenetrationPower`等字段是否存在且格式正确。

### Q3: 缺少高级属性（InitialSpeed等）？
**A**: 这些是可选属性，只有原始数据包含时才会添加。可以手动在生成的JSON中添加。

### Q4: 如何修改默认值？
**A**: 编辑脚本中的`DEFAULT_AMMO_TEMPLATE`字典：
```python
DEFAULT_AMMO_TEMPLATE = {
    "$type": "RealismMod.Ammo, RealismMod",
    "Name": "",
    "Damage": 50,  # 修改默认伤害
    "PenetrationPower": 20,  # 修改默认穿透
    "LoyaltyLevel": 1,
    "BasePriceModifier": 1
}
```
