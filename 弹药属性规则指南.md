# Realism 弹药数值规则指南

本文档用于说明《现实主义 (Realism)》模组中弹药（Ammo）补丁的自动生成规则。

规则实现文件：`ammo_rule_ranges.py`
规则应用入口：`generate_realism_patch.py` 中 `apply_realism_sanity_check()` 的 `RealismMod.Ammo` 分支。

---

## 1. 弹药物品（Ammo）必填基础字段

所有弹药物品数值对象建议至少包含以下基础属性：

- "$type": "RealismMod.Ammo, RealismMod"
- "ItemID": 物品 MongoID
- "Name": 内部引用名称
- "LoyaltyLevel": 商店解锁等级（1-4）
- "BasePriceModifier": 基础价格系数

---

## 2. 本规则覆盖字段

当前按“口径 + 穿深”分档重算以下 15 个字段：

- "InitialSpeed"
- "BulletMassGram"
- "Damage"
- "PenetrationPower"
- "ammoRec"
- "ammoAccr"
- "ArmorDamage"
- "HeatFactor"
- "HeavyBleedingDelta"
- "LightBleedingDelta"
- "DurabilityBurnModificator"
- "BallisticCoeficient"
- "MalfMisfireChance"
- "MisfireChance"
- "MalfFeedChance"

字段解释：

- "InitialSpeed"：弹头初速（m/s），影响弹道平直度与飞行时间；通常越高越容易命中中远距离目标。
- "BulletMassGram"：弹头质量（克），影响后坐感受、动能传递与下坠特征；重弹通常后坐更明显。
- "Damage"：基础肉体伤害，反映命中无甲/低护甲部位时的杀伤能力。
- "PenetrationPower"：穿透能力，决定对子弹穿甲表现与对高等级护甲的有效性。
- "ammoRec"：子弹后坐修正，数值越高通常代表该弹会带来更强后坐负担；负值代表后坐压力较小。
- "ammoAccr"：子弹精度修正，正值一般表示更稳定/更准，负值表示散布增大。
- "ArmorDamage"：护甲损伤系数（倍率），反映子弹对护甲耐久消耗速度；本规则使用小数系数，目标范围约为 1.00 到 1.50。
- "HeatFactor"：发热系数，影响武器升温速度；越高越容易在连续射击中积热。
- "HeavyBleedingDelta"：重出血倾向修正，值越高越容易触发重出血。
- "LightBleedingDelta"：轻出血倾向修正，值越高越容易触发轻出血。
- "DurabilityBurnModificator"：耐久烧蚀系数，影响枪械耐久损耗速度；高压/高穿弹通常更高。
- "BallisticCoeficient"：弹道系数（BC），反映飞行中保速与抗风能力；一般越高远距离速度衰减越慢。
- "MalfMisfireChance"：故障-哑火概率，反映弹药导致击发失败的基础概率。
- "MisfireChance"：通用哑火概率系数，与武器状态共同决定实战哑火出现频率。
- "MalfFeedChance"：故障-供弹异常概率，反映弹药导致卡弹/供弹失败的倾向。

故障概率字段规则补充：

- 常规弹药默认区间：`0.001 到 0.008`。
- 高压口径（如 `rifle_762x51` / `full_power_rifle` / `magnum_heavy` / `anti_materiel_50bmg`）上限可提高到 `0.015`。
- 高穿深档位（`pen_high` / `pen_very_high` / `pen_ap_extreme`）会对这 3 个字段施加正向增量。
- 叠加后统一硬限制在 `0.001 到 0.015`，避免越界。

说明：

- 第一层：按口径档位提供基础区间。
- 第二层：按穿深档位对第一层区间做增量修正（delta）。
- 最终值会在修正后的区间内进行加权采样，以保留原始输入风格并约束异常值。

## 3. 口径档位（第一层）

以下档位对应 `ammo_rule_ranges.py` 的 `AMMO_PROFILE_RANGES`：

### 3.0 EFT 手感子档（v2）

本次新增的“子档”会优先于大类匹配，用于强化 EFT 原版常见口径差异：

- `rifle_545x39`：低后坐、较高初速，偏稳定连发
- `rifle_556x45`：高初速、平衡后坐与穿深
- `rifle_762x39`：较高肉伤、后坐更明显
- `rifle_762x51`：全威力步枪弹，后坐与穿甲压力更高
- `rifle_9x39`：低初速重弹头，亚音速重击风格
- `rifle_300blk`：超/亚音速混合风格，参数跨度更大

子档关键词示例（完整见 `AMMO_PROFILE_KEYWORDS`）：

- `5.45x39` -> `rifle_545x39`
- `5.56x45 / .223` -> `rifle_556x45`
- `7.62x39` -> `rifle_762x39`
- `7.62x51 / .308` -> `rifle_762x51`
- `9x39` -> `rifle_9x39`

## 4. 穿深档位（第二层增量修正）

以下档位对应 `AMMO_PENETRATION_TIERS` 与 `AMMO_PENETRATION_MODIFIERS`：

- `pen_very_low`: 0 到 19
- `pen_low`: 20 到 29
- `pen_medium`: 30 到 39
- `pen_high`: 40 到 49
- `pen_very_high`: 50 到 64
- `pen_ap_extreme`: 65+

增量修正规则说明：

- 低穿深档位：通常提高伤害/流血倾向，降低穿甲与护甲伤害。
- 中穿深档位：保持平衡，接近口径基础区间。
- 高穿深档位：通常提高穿深/护甲伤害与热负担，降低肉伤与流血倾向。

注：最终区间 = 口径基础区间 + 穿深增量区间（按字段分别相加）。

---

## 5. 口径与穿深识别来源

脚本会从以下字段综合识别：

- 口径：`Caliber / AmmoCaliber / ammoCaliber / ammoType / Name`
- 穿深：`PenetrationPower`（若缺失则尝试 `Penetration / penPower`）

当口径无法识别时，默认使用 `intermediate_rifle`；
当穿深无法识别时，默认使用 `pen_medium`。

---

## 6. 调参建议

- 想让某口径整体更强/更弱：改 `AMMO_PROFILE_RANGES` 对应档位。
- 想调“同口径内 AP/HP 差异”：改 `AMMO_PENETRATION_MODIFIERS`。
- 想补充新口径识别：改 `AMMO_PROFILE_KEYWORDS`。
- 想改变穿深分层阈值：改 `AMMO_PENETRATION_TIERS`。
