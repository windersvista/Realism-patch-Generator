# 快速入门指南

## 🚀 3步开始使用

### 步骤 1: 准备物品数据
将需要生成补丁的武器和配件JSON文件放入 `input/` 文件夹。

### 步骤 2: 运行脚本
双击 `运行补丁生成器.bat` 文件，或在命令行运行：
```bash
python generate_realism_patch.py
```

### 步骤 3: 获取结果
在 `output/` 文件夹中查看生成的补丁文件。
输出文件名将根据输入文件名自动生成，例如：
- `1-WeaponVelociraptor.json` -> 原始数据
- `output/1-WeaponVelociraptor_realism_patch.json` -> 生成的补丁

## ✅ 基本检查

生成补丁后，请检查：

1. **文件已创建**: `output/` 文件夹中有3个JSON文件
2. **数量合理**: 生成的物品数量符合预期
3. **格式正确**: 打开JSON文件确认格式无误

## ❓ 常见问题

**Q: 脚本提示"未找到匹配的模板"**  
A: 这是正常的。某些物品类型（如弹药、任务道具）不需要现实主义补丁，会被自动跳过。

**Q: 生成的补丁数量比预期少**  
A: 查看控制台输出中的"跳过"信息，了解哪些物品被跳过及原因。

**Q: 如何调整某个武器的属性？**  
A: 直接编辑 `output/` 文件夹中生成的JSON文件，修改对应武器ID的属性值。
