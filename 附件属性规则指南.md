# Realism 武器附件数值规则指南

本文档旨在为《现实主义 (Realism)》模组的武器附件数值生成提供参考规则。这些属性直接从“现实主义物品模板 (attatchments)”文件夹中提取，专门用于武器附件（MOD）类物品，其数据结构与武器（Gun）完全不同。

---

## 1. 附件物品（WeaponMod）必填基础字段

所有附件物品（WeaponMod）数值对象必须包含以下基础属性：

- "$type": "RealismMod.WeaponMod, RealismMod"
- "ItemID": 物品的 MongoID
- "ModType":
- "Stock*"(附件类型：枪托)
- "stock_*_buttpad"(附件类型：枪托垫)
- "stock_adapter"(附件类型：枪托转接器)
- "sight"(附件类型：所有瞄具)
- "mount"(附件类型：可安装配件/带皮轨)
- "grip_stock_adapter"(附件类型：握把枪托转接器)
- "booster"(附件类型：枪口助推器)
- "muzzle_supp_adapter"(附件类型：可用作消声器转接器)
- "sig_taper_brake"(附件类型：Sig消声器专用转接器)
- "bayonet"(附件类型：刺刀)
- "foregrip_adapter"(附件类型：前握把专用转接器)
- "long_mag"(附件类型：长弹匣)
- "shotTube"(附件类型：霰弹枪弹仓)
- "shot_pump_grip_adapt"(附件类型：泵动霰弹枪前握把转接器)
- "gas"(附件类型：消声器/抑制器优化型拉机柄)
- "short_barrel"(附件类型：短枪管，长度在178 - 292mm范围内)
- "bipod"(附件类型：两脚架)
- "Weight": 物品的重量（kg）
- "LoyaltyLevel": 商店解锁等级（1到4）

## 2. 各分类属性解析 (现实主义标准)

### 所有配件可用属性的说明及示范值**

- "ReloadSpeed [更换弹匣速度，数值越高越快]": 5
- "LoadUnloadModifier [子弹压弹/退弹速度乘数，数值越高越快]": 20
- "CheckTimeModifier [检查弹匣速度乘数，数值越高越快]": 3
- "VerticalRecoil [数值越低越好]": 0,
- "HorizontalRecoil [数值越低越好]": -3,
- "Dispersion [数值越低越好，整体散布]": -15,
- "CameraRecoil [数值越低越好]": -10,
- "AutoROF [1代表1%射速提升]": 1,
- "SemiROF [2.5代表2.5%射速提升]": 2.5,
- "ModMalfunctionChance [数值越低越好]": -10,
- "CanCycleSubs [是否让无法循环亚音速弹的口径循环亚音速弹药,supressor/silencer必须true]": false,
- "Accuracy [数值越高越好]": -5,
- "HeatFactor [数值越高越差]": 1.13,
- "CoolFactor [数值越高越好]": 0.95,
- "DurabilityBurnModificator [数值越高越差]": 1.1,
- "Velocity [2=2%初速提升，如果是枪管，则使用同口径相近长度枪管的现实主义模组数据]": 2,
- "RecoilAngle [5=后坐角度增加5%以上，趋向90度（垂直向上）]": 5,
- "ConflictingItems [应冲突的物品，将与原冲突列表合并，而非覆盖]": [],
- "Ergonomics [数值越高越好]": 0,
- "Weight [重量，制作数值时使用输入文件的同属性值]": 0.354,
- "Loudness [负值表示更安静，用于致聋机制和SAIN模组]": -32,
- "Convergence [数值越高越好，响应更迅速，更不飘，枪口上扬和后坐爬升更小]": 0,
- "LoyaltyLevel[商店解锁等级（1-4），制作数值时使用输入文件的同属性值]": 3,
- "Flash [数值越高，若为消音器或非枪口装置则气体更多，否则火焰更明显]": 15,
- "Handling [武器在移动鼠标或行走/侧移时惯性阻力更小]": 6,
- "AimStability [武器瞄准稳定性，越高越好]": 7.5,
- "AimSpeed [越高越好]": 5,
- "StockAllowADS [覆盖被设定为阻止开镜的装备物品]": false,
- "HasShoulderContact [枪托是否实际接触玩家肩部]": true,
- "CenterOfImpact [影响枪管精度，数值越高精度越差]": 0.042,
- "ModShotDispersion [负值减小鹿弹散布]": -25

### 🚀 枪口装置 (MuzzleDevice)

*主要影响后坐力、隐蔽性及枪口状态。数值应根据功能细分属性。*

**消声器 (muzzle_suppressor)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 降低枪口噪声与冲击波，减轻射手及周边人员听觉负担。
- 压制枪口焰，降低夜间/低光环境下的可见暴露。
- 缓和主观后坐与爆震感，提升连续射击时的可控性。
- 改善队内沟通与环境感知，射手更容易听清口令与周边动静。
- 在室内、车辆内等反射环境中，减少声压反射造成的不适。
- 与亚音速弹配合时可进一步降低声学特征，提升隐蔽性。
- 注：消声器是“降噪”而非“无声”，仍会保留明显射击声。

- "Ergonomics": -18 到 -8
- "CameraRecoil": -8 到 -3
- "VerticalRecoil": -18 到 -10
- "HorizontalRecoil": -12 到 -6
- "Dispersion": -5 到 -1
- "Accuracy": -5 到 -1
- "Velocity": 0.2 到 1.0
- "Loudness": -40 到 -20
- "Flash": -80 到 -30
- "ModMalfunctionChance": 10 到 25
- "DurabilityBurnModificator": 1.2 到 1.5
- "AimSpeed": -20 到 -8

**消焰器 (muzzle_flashhider)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 核心目标是抑制枪口焰，优先降低可见暴露而非显著降噪。
- 通常对后坐控制有轻度改善，但弱于制退器/补偿器。
- 常用于夜战或低光环境，减少火焰对射手视野与夜视设备的干扰。

- "Ergonomics": -5 到 -2
- "CameraRecoil": -2 到 -1
- "VerticalRecoil": -5 到 -2
- "HorizontalRecoil": -3 到 -1
- "Dispersion": -2 到 2
- "Loudness": 0 到 10
- "Flash": -70 到 -40
- "AimSpeed": -3 到 -1

**制退器/补偿器 (muzzle_brake)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 通过导流火药燃气抑制后坐与枪口上跳，提升连发可控性。
- 代价通常是侧向冲击波和噪声更明显，对周边人员影响更大。
- 更偏向竞技/压枪场景，而非隐蔽射击场景。

- "Ergonomics": -8 到 -3
- "CameraRecoil": -10 到 -3
- "VerticalRecoil": -24 到 -14
- "HorizontalRecoil": -18 到 -10
- "Dispersion": -5 到 -2
- "Accuracy": -2 到 -1
- "Velocity": 0.5 到 1.0
- "Loudness": 10 到 20
- "Flash": 3 到 15
- "AimSpeed": -6 到 -2

**枪口保护器 (muzzle_thread)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 主要用于保护枪口螺纹，避免磕碰与污染导致装配问题。
- 对弹道和操控影响极小，通常作为默认占位或过渡件使用。
- 便于后续快速更换消音器、制退器等枪口装置。

- "Ergonomics": 0 到 1
- "CameraRecoil": 0 到 0
- "VerticalRecoil": 0 到 0
- "HorizontalRecoil": 0 到 0
- "Dispersion": 0 到 0
- "Accuracy": 0 到 0
- "Velocity": 0 到 0
- "Loudness": 0 到 0
- "Flash": 0 到 0
- "ModMalfunctionChance": 0 到 0
- "DurabilityBurnModificator": 1.0 到 1.0
- "AimSpeed": 0 到 0

**枪口转接口 (muzzle_adapter，含 muzzle_*_adapter / sig_taper_brake)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 作用是完成接口标准转换，让不同制式枪口装置可兼容安装。
- 通常性能增益有限，更多体现为“功能解锁件”。
- 设计上应控制代价较小，但可保留少量重量/人机惩罚。

- "Ergonomics": -2 到 0
- "Flash": -5 到 5
- "DurabilityBurnModificator": 1.0 到 1.1

### 🎒 弹匣 (Magazine)

*影响装填效率和供弹可靠性。*

当前实现按容量分为 4 档（见 `magazine_compact/standard/extended/drum`）：

**紧凑弹匣 (magazine_compact, <=20 发)**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 体积和重量小，换弹与举枪操作更轻快。
- 供弹行程短，可靠性通常较稳定。
- 代价是持续火力与容错较低。

- "Ergonomics": 0 到 6
- "ReloadSpeed": 0 到 8
- "LoadUnloadModifier": -8 到 0
- "CheckTimeModifier": -3 到 1
- "MalfunctionChance": -3 到 1
- "AimSpeed": 0 到 4
- "Handling": 0 到 5

**标准弹匣 (magazine_standard, 21-40 发)**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 在容量、重量、操控之间保持均衡，是大多数平台的基准配置。
- 适合通用任务，几乎没有明显极端短板。

- "Ergonomics": -4 到 1
- "ReloadSpeed": -4 到 3
- "LoadUnloadModifier": 0 到 8
- "CheckTimeModifier": 0 到 4
- "MalfunctionChance": -1 到 3
- "AimSpeed": -3 到 1
- "Handling": -3 到 1

**加长弹匣 (magazine_extended, 41-60 发)**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 以容量换取持续压制能力，减少换弹频率。
- 增重和重心下移会明显拖慢举枪、转向与机动。
- 高负载下供弹阻力增大，可靠性风险相对上升。

- "Ergonomics": -12 到 -3
- "ReloadSpeed": -12 到 -2
- "LoadUnloadModifier": 6 到 15
- "CheckTimeModifier": 2 到 8
- "MalfunctionChance": 1 到 6
- "AimSpeed": -6 到 -1
- "Handling": -6 到 -2

**鼓弹匣 (magazine_drum, >=61 发)**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 极高容量适合持续火力与定点压制。
- 重量、体积和操作惩罚最大，机动与换弹效率显著下降。
- 结构复杂，通常更容易出现供弹与维护问题。

- "Ergonomics": -22 到 -8
- "ReloadSpeed": -22 到 -8
- "LoadUnloadModifier": 12 到 25
- "CheckTimeModifier": 6 到 14
- "MalfunctionChance": 3 到 10
- "AimSpeed": -10 到 -3
- "Handling": -12 到 -4

### ⚙️ 导气箍 (Gasblock)

**导气箍 (gasblock)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 负责导气系统节律，影响武器循环速度与可靠性。
- 对后坐与温控影响通常较细微，但会持续作用于整枪状态。
- 偏“系统稳定性部件”，不应给出过度夸张的单项增益。

- "Ergonomics": -2 到 0

### 🎯 瞄准镜 (Scope)

- 更依赖稳定射击姿态，近距离反应性通常不占优。

- "AimSpeed": -5 到 -15 (倍率越大负值越高)

- "Ergonomics": -5 到 -12

**红点镜 (scope_red_dot)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 强调近中距离快速捕获目标与快速复位。
- 视野更开阔、遮挡更少，动态射击更友好。
- 放大能力有限，不适合远距离精确识别。

- "AimSpeed": 2 到 8
- "AimStability": 0 到 5
- "Ergonomics": 0 到 -3
- "Weight": 取输入文件中的同属性的值
- "LoyaltyLevel": 取输入文件中的同属性的值

### 🎯 机械瞄具 (IronSight)

*主要影响瞄准性能。*

**核心属性 (iron_sight)数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 机械结构简单、可靠性高，不依赖电池和额外光学组件。

- "LoyaltyLevel": 取输入文件中的同属性的值

### 🪵 枪托 (Stock)

*主要影响后坐力控制和稳定性。*

**固定枪托 (stock_fixed)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 结构刚性高，后坐传递稳定，利于连续精确射击。
- 稳定性强但灵活性差，通常带来更明显的机动惩罚。
- 适合中远距离和阵地化作战节奏。

- "VerticalRecoil": -15 到 -10
- "HorizontalRecoil": -10 到 -6
- "CameraRecoil": -20 到 -12

- "AimSpeed": -12 到 -6

- "Ergonomics": -15 到 -8

**折叠/伸缩枪托 (stock_folding)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 在稳定性与便携性之间折中，适应多场景转换。
- 折叠或短位有利于室内/车内操作与携行。
- 相比固定枪托，极限控枪能力通常略弱。

- "VerticalRecoil": -10 到 -5
- "HorizontalRecoil": -6 到 -2

现实作用归纳（用于理解该分类数值设计逻辑）：

- 重点提升开镜状态下的支撑与瞄准稳定性。
- 在维持一定操控性的同时，增强持续瞄准舒适度。
- 适合需要频繁开镜交战的中距离配置。

- "VerticalRecoil": -8 到 -4
- "HorizontalRecoil": -5 到 -2

- 提供后托安装基础并参与后坐行程相关结构支撑。
- 对整枪操控影响通常温和，更多体现为平台适配价值。

**枪托适配器（stock_adapter）属性数值范围**:

- 可小幅提升稳定性与舒适度，通常伴随轻微增重。

### 🔫 枪管 (Barrel)

*决定武器的精度、初速和发热表现。*

**短枪管 (barrel_short)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：

- 枪身更紧凑，转向与近距离操作更灵活。
- 初速和远距精度通常下降，热负荷与磨损压力更高。
- 适合高机动、短接触距离场景。

- "Velocity": -10 到 -2
- "Accuracy": -5 到 -20
- "HeatFactor": 1.1 到 1.3

- "DurabilityBurnModificator": 1.1 到 1.3
- "RecoilAngle": 5 到 15
- "Weight": 取输入文件中的同属性的值
- "LoyaltyLevel": 取输入文件中的同属性的值

现实作用归纳（用于理解该分类数值设计逻辑）：

- 在机动性、初速、精度和温控之间提供均衡表现。
- 适用范围最广，通常作为默认基准长度。

- "CenterOfImpact": 0.02 到 0.05
- "Velocity": -2 到 5
- "Accuracy": -2 到 5

- "Weight": 取输入文件中的同属性的值
- "LoyaltyLevel": 取输入文件中的同属性的值

**长枪管 (barrel_long)属性数值范围**:

现实作用归纳（用于理解该分类数值设计逻辑）：
